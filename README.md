# Sheriff Jones Robber Roundup

A simple arcade game built with Python and the Arcade library. Control Sheriff Jones to catch moving robbers!

## Description

Robbers move back and forth across the screen. Use the arrow keys to move the sheriff and catch them. The game features a basic GUI for starting and instructions.

## Requirements

- Python 3.x
- Arcade library
- PyQt5

## Installation

1. Install dependencies:
   ```
   pip install arcade PyQt5
   ```

## How to Run

Run the game from the project root:
```
python src/SimpleGame.py
```

## Controls

- Arrow keys: Move sheriff up, down, left, right
- P: Pause/unpause the game
- R: Restart after game over
